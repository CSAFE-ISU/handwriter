R Package for Handwriting Analysis

A detailed tutorial on using handwriter is available on the How To page.

The handwriter package can be installed from CRAN using: 

                      install.packages("handwriter")
                        library(handwriter)

How It Works

A step by step explanation on how handwriter works is available on the Methods page. The package is open-source and the code can be viewed on Github.

Information on integrating handwriter into your workflow is available on the Workflow Integration page.

Research

handwriter is part of the Handwriting Evaluation project by CSAFE at Iowa State University.

More information on the entire project can be found on the Research page.
